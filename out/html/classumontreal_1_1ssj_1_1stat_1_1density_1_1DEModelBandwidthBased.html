Detailed Description

Recall that asymptotically, in both cases, \(\textrm{ISB} \approx B h^{\alpha}\).

Now, for MC and hence for RQMC too, it is known (see [189]) that \( B = R(f')/12 \) for histograms and \(B = \mu_2(k)R(f'')/4 \) for KDEs, where

\[ R(g) = \int_a^b g^2(x)\mathrm{d}\x\]

denotes the roughness functional (see umontreal.ssj.stat.DensityEstimator::roughnessFunctional(double[], double, double) ) and where \(\mu_k \) denotes the \(k\)th moment \(\int_{-\infty}^{\infty}x^k g(x)\mathrm{d}x \). Thus, in order to estimate \(B\) one needs to estimate derivatives of the sought density.

Computes the asymptotically optimal multiplicative constant in the ISB for a histogram as described above.

The bandwidth in these density derivate estimates is computed recursively via hOptAmise(DensityDerivativeEstimator, double[], double, double[], double[], int, double) starting with initial value init for the roughness functional of the derivative of the density of order maxDerivative. Each occuring integration, e.g. for evaluating a roughness-functional, is carried out by a quadrature rule using the integration nodes evalPoints.

Analogously to hOptAmise(DensityDerivativeEstimator, double[], double, double[], double[], int, double) mu2 is the second moment of the kernel and mu2Derivatives contains the second moments of the derivatives of \(k\) decreasing w.r.t. the order of the derivatives, starting with order maxDerivative - 2.

Note that for a histogram estimator maxDerivative has to be odd!

Parameters
ddethe DDE we use for determining the optimal bandwidth.
datathe observations gained from \(m\) independent simulations.
mu2the second moment of \(k\).
mu2Derivativesthe second moments of the derivatives of \(k\).
evalPointsthe integration nodes.
maxDerivativethe highest order of the density derivative considered.
initinitial value of the roughness functional in the denominator.
dethe histogram estimator for the sought density.
Returns
the asymptotically optimal multiplicative constant in the ISB.

◆ computeIVandISBandMISE() [1/2]

double [] computeIVandISBandMISE ( double  estDensities[][])

This method estimates the IV, ISB, and MISE based on \(m\) realizations of the density estimator, which have previously been evaluated at the \(k\) points stored in evalPoints, and returns them in an array in this order.

The matrix estDensities has dimensions \(m\times k \), i.e. each row contains the evaluations of one density estimator.

The estimate for the IV is obtained from an estimate of the empirical IV computeIV(double[][]), the ISB is estimated by computeEstimatedISB(), and the MISE-estimate is obtained by their sum.

Parameters
estDensitiesthe \(m\times k\)matrix containing the results of evaluating \(m\) densities at \(k\) evaluation points.
Returns
estimates for the IV, the ISB, and the MISE.

Depending on whether maxDerivative is even or odd, this method computes \( h_{\text{AMISE}}^{(r_0)}\) for \(r_0=2\) or \(r_0=1\), respectively.

This is done by recursively applying the formula ( hopt ) from above. The recursion starts at \(r=\textit{maxDerivative} - 2\) with the initial value init for \(R(f^{(r+2)})\). The second moment of \(K\) is given by mu2 and the second moments of the derivatives \(k^{(r)},k^{(r-2)},\dots,k^{(r_0)} \) are passed via mu2Derivatives. It is assumed that the elements of mu2Derivatives are sorted by order of derivative in a decreasing manner, the first one being \(\mu_2(k^{(r)})\).

Parameters
ddethe DDE to use in the recursion.
dataobservations from one simulation.
mu2the second moment of \(k\).
mu2Derivativesthe second moments of the derivatives of \(k\).
evalPointsthe integration nodes.
maxDerivativethe highest order of the density derivative considered.
initinitial value of the roughness functional in the denominator.
Returns
either \( h_{\text{AMISE}}^{(2)}\) or \( h_{\text{AMISE}}^{(1)}\), depending on whether maxDerivative is even or odd.
