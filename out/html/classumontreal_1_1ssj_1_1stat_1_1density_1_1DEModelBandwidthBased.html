Detailed Description

Implements a parametric model that is particularly designed for the estimation of the density of a random variable \(X = g(\mathbf U)\), where \(\mathbf U = (U_1,U_2,\dots,U_d)\sim U(0,1)^d \) with a given function \(g:(0,1)^d\rightarrow\mathbb{R}\) over a finite interval \([a,b]\).

Such situations arise naturally in uncertainty quantification. The observations \(X_0,X_1,\dots,X_n\) which are used to construct the density will typically be generated by simulation of \(\mathbf U\), either by Monte Carlo (MC) or by randomized quasi-Monte Carlo (RQMC).

The model applies to density estimators that rely on the selection of a bandwidth (or binwidth) such as histogram estimators, see DEHistogram, and kernel density estimators (KDEs), see

DEKernelDensity, for instance. It is thoroughly investigated in (TODO: cite our paper) and can be explained as follows. It is known in general, that the mean integrated square error (MISE) can be rewritten as the sum of the integrated variance (IV) and the integrated square bias

\[ \textrm{MISE} = \int_a^b\mathbb{E} [\hat{f}_{n,h}(x) - f(x)]^2\mathrm{d}x = \int_a^b\textrm{Var}[\hat{f}_{n,h}(x)]\mathrm{d}x + \int_a^b \left( \mathbb{E}[\hat{f}_{n,h}(x)] - f(x) \right)^2\mathrm{d}x = \textrm{IV} + \textrm{ISB}, \]

where \(f\) denotes the true density and \(\hat{f}_{n,h}\) the density estimator. For MC it is known that, asymptotically, \(\textrm{IV}\approx C n^{-1} h^{-1}\) and \(\textrm{ISB}\approx B h^{\alpha}\), \(C,B,\alpha>0\). Observe that the power of \(h\) is positive in the ISB, while it appears to be negative in the IV.

The goal is to select an optimal band- or binwidth \(h_*\) which balances these two terms, which is also known as the infamous variance-bias tradeoff. This is possible, as soon as all the parameters \(C,B,\alpha\) are explicitly known (or can be sufficiently well estimated, at least). For histograms and KDEs this is indeed the case. Let \(R(g)\) denote the roughness functional over the interval \([a,b]\) and let \(\mu_k(g)\) be the \(k\)-th moment of a function \(g\), i.e.

\[ R(g) = \int_a^bg^2(x)\mathrm{d}x,\qquad\text{and}\quad \mu_k(g)=\int_{-\infty}^{\infty}x^kg(x)\mathrm{d}x. \]

Assuming that all the entities exist and are finite, the table below lists the parameter values for histograms and KDEs for observations generated with MC, see [189].

Asymptotically optimal parameter values with MC
Parameter Histogram KDE
\(C\) 1 \(\mu_0(K^2)\)
\(\alpha\) 2 4
\(B\) \(R(f')/12\) \( \mu_2^2(K) R(f'')/4\)

The only highly unpleasant quantities are the roughness functional of the first or second derivative of the unknown density. This, however, can be overcome by using derivative estimates as implemented in DensityDerivativeEstimator and plugging the results back into the expression for the ISB. The kernel assumed for the derivative estimation is a standard normal. To estimate the \(h^{r_0}_{\text{AMISE}} \), where \(r_0\) can be seen in the table above, we use one step of the recursion DensityDerivativeEstimator::hAmiseR(int, int, double, double[], double, DensityDerivativeEstimator, double[], double[], double, double). For the initial value we estimate the empirical standard deviation \(\hat{\sigma}\) from the observations \(X_0,\dots,X_{n-1}\) and assume \(f\) to be a normal distribution with standard deviation \(\hat{\sigma}\).

If the user wishes to use different kernels or other initial values than those stated above, it is only necessary to overwrite the corresponding methods for estimating \(B\). The rest will remain intact.

Let us now turn to the IV. What changes with RQMC is that it can cause significant variance reduction, which, however, can come at price of a worse dependency on \(h\). This was proven in (TODO: cite our paper) theoretically and also observed empirically. To reflect this behavior, we choose the model

\[ \textrm{IV}\approx C n^{-\beta}h^{-\delta},\qquad \beta,\delta>1, \]

locally, i.e. within a small region w.r.t. \((n,h)\). This proved to be a reasonable assumption.

Observe that taking the logarithm of both sides of the above model assumption yields a linear model with variables \(\log n\) and \(\log h\), and unknown parameters \(\log C,\beta,\delta\). Based on this linear model, one can estimate the empirical IV for several reasonable values of \((n,h)\) with the methods provided by DensityEstimator and subsequently obtain estimates for the unknown parameters. Together with the estimated parameters from the ISB, one can thus compute an estimate for \(h_*\).

Finally, one can test the estimator with the selected \(h_*\) out of the box, estimate its empirical IV, compute its modeled ISB and add these quantities to obtain an estimator for the empirical MISE.

String parametersIVEstimate ( MonteCarloModelDouble  model,

This method is the core function for estimating the IV parameters \(C\), \(\beta\), and \(delta\) for a histogram estimator by estimating the empirical IV over a testing region for \((n,h)\) and, subsequently, performing a linear regression.

More precisely, rqmcPts is intended to contain the same type of point set in different sizes \(n\). These sizes are the \(n\)-values for the testing region. The \(h\)-values are passed in hArray. For every point in rqmcPts it generates observations of model. This is repeated \(m\) times independently. On the basis of these observations it estimates the empirical IV for every \(h\) in hArray. The evaluation points used in this step are the number of bins currently set for the histogram, as this is sufficient. The estimated IVs are stored in logIV3D. Finally, this data is used to estimate the parameters by a linear regression.

The string which is returned is composed of an introductory heading, a table showing the results of the estimates of the empirical IV for different \(n\) and \(h\), and a formatted output of the estimated parameters \(C\), \(\beta\), and \(delta\) so that the details of this experiment can be written to a file. If the user wishes to display output at runtime as well, simply set displayExec to true<>.

Parameters
rqmcPtsthe RQMC point sets to construct the observation.
modelthe underlying model, from which the observations are obtained.
mthe number of independent repetitions.
dethe histogram.
hArraythe values for \(h\) of the testing region.
Returns
a formatted String containing all the information about this parameter estimation.
