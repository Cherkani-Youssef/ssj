Detailed Description

This class implements the CDE for a sum of \(d\) normals with mean 0, which is normalized to have variance 1.

The realizations of the random variables at which this CDE can be obtained with the class #MultiNormalIndependent.

This estimator consists of \(d\) individual estimators \(\hat{f}_{j}\), \(1 \leq j \leq d\), where the index indicates which summand is hidden. For the final estimator we take a convex combination of the above estimators, i.e.,

\[\hat{f} = \sum_{j=1}^d \omega_j\hat{f}_{j},\]

where the \(\omega_j\) are positive weight whose sum is one.

If one wants to omit a certain estimator, one can set the corresponding \(\omega_j\) to a negative number. In this case the weight will be treated as zero. In this case, the remaining weights should sum up to one.
