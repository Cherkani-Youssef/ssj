Detailed Description

This class provides methods to compute a kernel density estimator (KDE) for univariate unimodal densities over an interval \([a,b]\) from a set of \(n\) individual observations \(X_0, â€¦, X_{n-1}\), and returns its value at a set of selected points.

This estimator takes a fixed bandwidth \( h>0\) as well as a kernel function \(K\), which is also referred to as kernel density. The kernel density should be non-negative and integrate to one. For \(x\in[a,b]\), the KDE itself is defined as

\[ \hat{f}_{n}(x) = \hat{f}_{n,h}(x) = \frac{1}{nh} \sum_{i = 0}^{n-1} k\left( \frac{x - x_i}{h} \right). \]

double evalDensity ( double  x)

Returns the value of the density evaluated at point \(x\).

For this evaluation method to work we assume that kernel is unimodal, i.e. increasing then decreasing. This algorithm works by summing up all contributions of all observations \(X_0, \ldots, X_{n-1}\) which are larger than eps.

◆ evalDensity() [2/2]

void evalDensity ( double []  evalPoints,
double []  density 
)

Returns in array density the value of the estimator at the evaluation points in evalPoints.

These two arrays must have the same size and needs to be sorted in increasing order.

This algorithm is more efficient than the default implementation in that it keeps track of which observations are already too far to the left of the current evaluation point to contribute more than eps to the estimated value. Avoiding the overhead of computing such negligible contributions can significantly reduce computational cost.
