Detailed Description

This class provides methods to compute a kernel density estimator (KDE) for univariate unimodal densities over an interval \([a,b]\) from a set of \(n\) individual observations \(x_0, â€¦, x_{n-1}\), and returns its value at a set of selected points.

This estimator takes a fixed bandwidth \( h>0\) as well as a kernel function \(K\), which is also referred to as kernel density. The kernel density should be non-negative and integrate to one. For \(x\in[a,b]\), the KDE itself is defined as

\[ \hat{f}_{n}(x) = \hat{f}_{n,h}(x) = \frac{1}{nh} \sum_{i = 0}^{n-1} k\left( \frac{x - x_i}{h} \right). \]
