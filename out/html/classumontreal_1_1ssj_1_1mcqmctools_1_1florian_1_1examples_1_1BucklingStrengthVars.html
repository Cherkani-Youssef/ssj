Constructor to pass the parameters of the distributions for all random variables involved.

This class is intended to create and return several random variables related to the buckling strength \(X\) of a steel plate as given by Schields and Zhang '16.

This is used for estimating the density of \(X\) with a conditional density estimator, for instance. The model is given by

\[ X=\left( \frac{2.1}{\lambda}-\frac{0.9}{\lambda^2} \right) \left( 1-\frac{0.75\delta_0}{\lambda}\right) \left(1-\frac{2\eta t}{b} \right), \]

where \lambda = (b/t) \sqrt{\sigma_0/eta} and where the parameters \(b,t,\sigma_0,E,\delta,\eta\) are random variables. More precisely, $t$ and $b_0$ follow a lognormal distribution and all the others are normally distributed.

The random variates that are returned by this class (in the stated order) are \(2t/b, \lambda, \delta_0, \eta\).
