Detailed Description

This class implements a conditional density estimator for the displacement of a cantilever beam.

\(f(E,X,Y)\) as explained in https://www.sfu.ca/~ssurjano/canti.html. More precisely,

\[ f(E,X,Y) = \frac{4L^3}{Ewt}\sqrt{ \frac{Y^2}{t^4} + \frac{X^2}{w^4} }, \]

where \(L,w\) and \(t\) are constants. The random variables \(E,X,Y\) are assumed to be normally distributed and independent.

In fact, this class derives three estimators (each variable is hidden once), and consider their weighted sum as the CDE. Weights are supposed to be positive and sum up to one. If a negative weight is passed, it is automatically set to zero.
