Computes the asymptotically optimal multiplicative constant in the ISB for a histogram.

This works exactly as computeB(DensityDerivativeEstimator, double[][], double, double[], double[], int, double, DEHistogram) but with the kernel of the DDE being a Gaussian.

In this case, the second moment of the kernel is 1 and those of the derivatives can be obtained via densFunctionalGaussian(int, double).

Note that for a histogram estimator maxDerivative has to be odd!

dethe histogram estimator for the sought density.

the asymptotically optimal multiplicative constant in the ISB.

◆ densityFunctionalGaussian()

static double densityFunctionalGaussian ( int  r,
double  sigma 
)
static

Computes \(\Phi_{2r}(p)\), i.e.

the density functional of order \(2r\) of a normal density \(p\) with standard deviation sigma,

\[\Phi_{2r}(p) = \int_{\mathbb{R}} p^{(2r)}(x) p(x)\mathrm{d}\!x\]

and it can be comuted as

\[\Phi_{2r}(p) = \frac{(-1)^r (2r)!}{(2\sigma)^{2r+1} r! \sqrt{\pi}}. \]

It is worth mentioning that the second moment of the \(r\)-th derivative of any density \(g\) is related to \(\Phi_{2r}(g)\) via \(\mu_2(f^{(r)}) = (-1)^r \Phi_{2r}(f)\).

Parameters
rthe order of the functional. Has to be even.
sigmathe standard deviation of the normal density considered.
Returns
the density functional of order \(2r\).
