Detailed Description

This abstract class takes multi-dimensional objects such as arrays or #MultiDim's, maps them to the real numbers via a scoreFunction \(h\) and then sorts the objects according to their value under \(h\).

The score function is individual for each setting and is thus implemented as an abstract method.
