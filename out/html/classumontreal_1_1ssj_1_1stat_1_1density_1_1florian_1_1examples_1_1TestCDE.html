Detailed Description

This class provides all necessary tools as well as a main() function to run an experiment for estimating densities with a conditional density estimator (CDE) or a log-likelihood estimator (LR).

Typically, this will require that a #MonteCarloModelArray is provided together with a pertinent #ConditionalDensityEstimator.
