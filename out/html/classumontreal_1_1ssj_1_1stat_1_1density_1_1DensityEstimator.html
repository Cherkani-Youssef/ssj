Detailed Description

This abstract class implements a univariate density estimator over a finite interval \([a,b]\).

The density is constructed from a set of \(n\) observations. Some of these observations can lie outside the interval \([a,b]\).

The density can be evaluated at a single point \(x\) or on a grid of evaluation points.

Furthermore, this class implements basic routines for the computation of the integrated variance (IV) as well as for calculating basic error measures, such as the integrated square bias (ISB) and the mean integrated square error (MISE) in case the true density is known.

Computes the Coefficient of determination \(R^2\) of the observed data data and the estimated data dataEstimated.

For observed data \(y=(y_1,y_2,\dots,y_n)\) and estimated data \( f=(f_1,f_2,\dots,f_n)\) this is defined as

\[ R^2 = 1 - \frac{\textrm{SS}_{\text{res}}}{\textrm{SS}_{\text{tot}}} \]

, where \( \textrm{SS}_{\text{res}} \) denotes the sum of squares of the residuals

\[ \textrm{SS}_{\text{res}} = \sum{i=1}^n (f_i - y_i)^2 \]

and where {SS}_{{tot}} is the total sum of squares

\[ \textrm{SS}_{\text{tot}} = \sum{i=1}^n (y_i - \bar{y})^2 \]

. The closer this quantity is to one, the better the approximation of $y$ by $f$.

Computes all traits for the density estimator that are specified in the list traitList.

The density estimator itself is constructed from the observations in data and the evaluation points to compute the traits are given in evalPoints. Possible choices of traits are "isb", "iv", and "mise", with deviations due to letter-capitalization considered.

Since this method takes a model of type MonteCarloModelDensityKnown, all the traits are computed empirically instead of merely estimated. This makes it particularly useful for testing estimators with toy models.

If traitsList contains only 1 element, then this method calls the usual method to compute the respective trait. Otherwise, it computes the traits in parallel. This leads to

The computed traits are returned in an array in the same order as they appear in traitsList.

datamatrix containing the observations of \(m\) independent repetitions.
