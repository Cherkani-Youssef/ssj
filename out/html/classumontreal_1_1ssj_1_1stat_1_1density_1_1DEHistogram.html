Detailed Description

This class provides methods to compute a histogram density estimator for univariate densities over an interval \([a,b]\) from a set of \(n\) individual observations \(X_0, â€¦, X_{n-1}\), and returns its value at a set of selected points.

For a fixed number of bins \(m>0\) we partition the interval \([a,b]\) into \(m\) subintervals of equal lengths \(h\). Observe that \(h=(b-a)/m\) and, hence, Histogram estimators can also be parametrized by a bandwidth \(h>0\), as long as the resulting number of bins is an integer. The estimator itself is defined by

\[ \hat{f}_{n}(x) = \hat{f}_{n,h}(x) = \frac{n_j}{nh},\quad\text{for } x\in[a+(j-1)h, a+jh), \quad j=1,\dots,m, \]

where \(n_j\) denotes the number of observations that fall in this interval.

Note that, due to the fact that the partition into \(m\) bins relies on half-open intervals, the boundary \(b\) is not included in any of these intervals. Since the probability of an observation being exactly equal to \(b\) is zero, we can effectively ignore this subtlety.
