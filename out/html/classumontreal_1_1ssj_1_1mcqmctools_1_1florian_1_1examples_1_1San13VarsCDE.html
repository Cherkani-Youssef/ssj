Detailed Description

With this class one can generate several random variates involved in the estimation of the longest path of a specific stochastic activity network with 9 nodes and 13 links, taken from Elmaghraby (1977) and used again in L'Ecuyer and Lemieux (2000), "Variance Reduction via Lattice Rules".

The random variables that are generated by this class correspond to the lengths of certain paths that remain when one considers a conditional estimator. This can, e.g., be used for estimating the density of the longest path with a conditional density estimator.
