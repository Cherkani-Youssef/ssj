Detailed Description

Implements an abstract class for density estimators that rely on the selection of a bandwidth \(h\) such as, e.g., DEHistogram or DEKernelDensity.

Furthermore, it provides basic methods such as to estimate the integrated variance (IV), for instance.

The IV of such estimators usually locally follow the model \(\textrm{IV}\approx C n^{-\beta}h^{-\delta}\) and their integrated square bias (ISB) is of the form \(\textrm{ISB}\approx B h^{\alpha}\), where \(n\) denotes the number of data points used to construct the density.

◆ computeDensityTraits() [1/4]

double [] computeDensityTraits ( ArrayList< String >  traitsList,
double  data[][],
double []  evalPoints 
)

Computes all traits for the bandwidth based density estimator that are specified in the list traitList.

The possible choices of traits are "isb", "iv", "estimatediv", "mise" and "estimatedmise", with deviations due to letter-capitalization considered.

Note that in order to be able to compute the traits, several parameters of the density estimator need to be known and set in advance. Conversely, if only the ISB is to be computed, data and evalPoints can be empty.

Parameters
traitsListthe traits which shall be computed.

The computed traits are returned in an array in the same order as they appear in traitsList.

Parameters
datamatrix containing the observations of \(m\) independent repetitions.
evalPointsthe points at which the density estimator is evaluated to compute the traits.
Returns
the values of the specified traits.
